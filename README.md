# SSClass - Frontend Practice and Interview Prep Repository

This repository is a collection of frontend practice files created while learning and teaching core web development topics.
## What This Repository Contains

- Topic-wise practice files (`.html`, `.js`) for quick concept revision.
- Interview-focused examples in CSS and JavaScript.
- Basic to intermediate mini-projects.
- Problem-solving files (array, number, object-based questions).
## Folder Structure (Overview)

### `Boots/`
Bootstrap practice files:
- Containers, grid, columns, gutters
- Layout-based pages like quiz, certificate, and frame-style templates

### `Css/`
CSS concept files:
- Rounded corners, border image, multiple backgrounds
- Gradients, shadows, text effects, transforms (2D/3D)
- Responsive examples and animation
- Includes an `InterView/` subfolder with interview-oriented CSS practice
### `jQuery Tutorial/`
jQuery practice:
- Hide/show, slide/fade effects
- Click, blur/focus, and form events
- `jquery2/` contains many additional numbered practice files (`JQ1` to `JQ40` style)

### `js/APi/`
Asynchronous JavaScript and API handling:
- Callback, callback hell
- Promise basics and chaining
- Async/await
- Fetch, GET, POST, PUT examples

### `js/Core/`
JavaScript fundamentals:
- `let/var/const`, data types, operators, conditionals, loops
- Strings, arrays, objects, map/filter/reduce
- functions, HOF, closures, events, try-catch

### `js/Dom/`
DOM manipulation practice:
- Query selectors, properties, class list
- Create/append operations
- Event handling, responsive menu concepts

### `js/Oops/`
Object-Oriented JavaScript:
- Objects, prototype, class
- Constructor, inheritance, super, assignments

### `js/DSA/`
Data Structures and algorithm basics:
- Stack, queue
- Singly linked list, doubly linked list
- Supporting JS files for custom implementations

### `js/Quection/`
Question/problem files:
- Armstrong, palindrome, prime, fibonacci
- array duplicate, reverse, max/min, two-sum
- object/array problem-solving snippets

### `js/Project/`
Mini frontend projects:
- Currency Converter
- Rock Paper Scissors
- Password Hide/Show
- Tic Tac Toe
- Todo List
- Restaurant UI demo

### `img/`
Common image assets used in examples/projects.

## Who Can Use This Repo

- Beginners learning frontend step by step
- Students preparing for interviews
- Anyone who wants quick examples of JS/CSS/DOM/jQuery topics

## How To Run

This is a static practice repository.

1. Clone/download the repository.
2. Open the folder in VS Code.
3. Open any `.html` file in browser (or use Live Server extension).
4. For `.js` files, run in browser console or Node.js where applicable.

## Notes

- File names are intentionally topic-based and classroom-style (many are numbered for sequence learning).
- Some file names contain spelling variations because they were created during live practice sessions.
- This repo is focused on learning and examples, not a single production app.

## Suggested Learning Order

1. `Css/` and `Boots/`
2. `js/Core/`
3. `js/Dom/`
4. `jQuery Tutorial/`
5. `js/APi/`
6. `js/Oops/` and `js/DSA/`
7. `js/Project/` and `js/Quection/`

## Contributing

If you want to contribute:

1. Keep topic-wise naming clear.
2. Add short comments where logic is complex.
3. Prefer small, focused practice files.
