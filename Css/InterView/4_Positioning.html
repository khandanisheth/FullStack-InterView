1️⃣ Static Position

Position: STATIC
• Default position
• Normal document flow
• top / left work nahi karta

2️⃣ Relative Position

Position: RELATIVE
• Apni original jagah se move
• Space reserve rahta hai
• top: 20px, left: 30px

3️⃣ Absolute Position

Parent Container
position: relative (reference)
Position: ABSOLUTE
• Parent ke relative
• Flow se bahar
• top: 50px, left: 60px

4️⃣ Sticky Position

Position: STICKY
• Scroll par chipak jaata hai
• top: 0

Scroll down to see sticky effect

5️⃣ Fixed Position

Scroll page — fixed box hamesha screen par rahega

Position: FIXED
• Viewport ke relative
• Scroll par move nahi karta
