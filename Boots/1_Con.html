.container

Fixed width on all devices (according to screen size)

.container-fluid

Always 100% width (full screen) on all devices

.container-sm

Full width until screen is 576px, after that fixed width

.container-md

Full width until screen is 768px, after that fixed width

.container-lg

Full width until screen is 992px, after that fixed width

.container-xl

Full width until screen is 1200px, after that fixed width

.container-xxl

Full width until screen is 1400px, after that fixed width
